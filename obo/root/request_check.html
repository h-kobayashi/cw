  3. 「ユーザー」とは、本サービスを利用することによって求職を行う個人をいいます。

第3条（当社）

第4条（ユーザーおよび病院等の禁止事項）

ユーザーおよび病院等は、本サービスの利用において以下の各号の行為を行ってはならないものとします。

  1. ユーザー、病院等、当社その他第三者を誹謗、中傷、脅迫する行為またはそのおそれがある行為。

  4. 本サービスを利用した営業活動または営利を目的とする情報提供等の行為。

    2. 突発的な本サイトや本サービスを提供するためのシステム等の故障等が発生した場合

当社は総合人材サービスの企業の果たすべき重要な企業責任として、個人情報保護に関する法規制に基づき、次のとおり「個人情報保護方針」を策定し、当社の役員、従業員のすべてがこの「個人情報保護方針」を理解し、個人情報の一層の保護に対して継続的な改善に努めてまいります。

1.個人情報の収集、利用、提供
個人情報保護の重要性を強く認識し、当社の事業目的の範囲内でのみ個人情報を収集、利用、提供および預託を行います。

2.個人情報への安全管理措置の実施
個人情報への不正なアクセス、個人情報の紛失、破壊、改ざんおよび漏えいなどを防止するため、合理的な安全対策および是正、予防措置を講じます。

3.個人情報に関する法令および規範などの遵守
個人情報に関する法令およびこれに基づく各種ガイドラインなどの規範を遵守し、かかる法令などを遵守するための個人情報保護規定、その他のコンプライアンス・プログラムを策定し、遵守します。

4.コンプライアンス・プログラムの継続的改善の実施
